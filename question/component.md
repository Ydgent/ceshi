### 常用组件
标题前面的小图标 
```
<link rel='icon' type='image/x-icon' href='xxx.ico'>
<link rel="shortcut icon" href="favicon.ico" type="image/x-icon" />
```
网页头部信息
```
<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
<title>这里是title</title>
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<link rel="shortcut icon" href="/Content/img/orange-logo.png" type="image/x-icon" />
<meta name="channel" content="Smartphone" />
<meta name="keywords" content="互联网理财平台，互联网金融，P2P投资理财，P2P网贷平台">
<meta name="description" content="这里是网站描述" />
<meta name="viewport" content="width=device-width,initial-scale=1,minimum-scale=1,maximum-scale=1,user-scalable=no" /> // 移动端
```
